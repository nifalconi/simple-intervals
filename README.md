# Simple Intervals

[![License: CC BY-NC-SA 4.0](https://img.shields.io/badge/License-CC%20BY--NC--SA%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-nc-sa/4.0/)

A tiny, calm interval timer for the things you actually do — workouts, meditation, focus blocks, stretches, cooking. Build a routine once, hit play, listen for the cues.

No accounts. No ads. No cloud. Everything you save stays on your phone. Works offline after the first visit, and you can install it to your home screen like a real app.

## What it does

Pick a routine, press play, and Simple Intervals walks you through it task by task. A big timer tells you how long is left. A gentle sound lets you know when to switch. If you like, it can pop an OS notification for each new task so you can look away without missing a beat.

When you're done, you get a little celebration screen and the run is logged to your history.

## Features

### Routines you can shape

- Stack any number of timed tasks into a routine.
- Name each task, drag to reorder, delete, add new ones.
- Optional rest between tasks (adjust the length).
- Four starter routines out of the box: HIIT Blast, Pomodoro focus, Meditation, Morning stretch.

### A runner that stays out of your way

- Two running styles: **Focus** (one big card for the current task, next peek below) or **List** (all tasks stacked, each disappearing as it finishes).
- Two timer animations: **Consume** (color fills in) or **Drain** (color recedes).
- Controls for pause, previous, restart task, skip, restart whole routine.
- Tap the card to pause mid-task.

### Sound cues — no files, no downloads

- Four sound packs: Classic beep, Gentle chime, Arcade blip, Boxing bell.
- Toggle per routine: start cue, halfway ping, end cue, 3-2-1 countdown.
- Master sound switch in Settings if you want quiet.

### OS notifications

- Flip on "Task notifications" inside a routine and your phone pops a notification for every new task — handy when the screen is face-down or you're across the room.

### History

- Last 50 sessions saved. Total time, tasks completed, session count at a glance.
- Clear it whenever you want.

### Feels like your app

- Light, dark, or auto theme.
- Five accent colors: Green, Sun, Coral, Sky, Violet.
- Three typography choices: Grot, Serif, Mono.
- Installable as a Progressive Web App — adds a home-screen icon, runs in its own window, works offline.

## The screens

- **Library** — your saved routines, tap one to edit or press play to start.
- **Editor** — name the routine, add tasks, set durations, tune per-routine sound and notifications.
- **Runner** — the live timer.
- **History** — what you finished and when.
- **Settings** — theme, accent, font, display style, running view, sound master switch, install prompt.

## Install it on your phone

Simple Intervals runs in the browser, but it feels much nicer as an installed app — home-screen icon, own window, works offline, no browser chrome in the way.

### iPhone / iPad (Safari)

1. Open the app in **Safari** (not Chrome — iOS only allows installs from Safari).
2. Tap the **Share** button (square with an arrow pointing up).
3. Scroll down and tap **Add to Home Screen**.
4. Confirm the name and tap **Add**. The icon lands on your home screen.

### Android (Chrome, Edge, Samsung Internet)

1. Open the app in Chrome (or any Chromium browser).
2. Either tap the **Install** prompt at the bottom of the screen, or open the browser menu (⋮) and choose **Install app** / **Add to Home screen**.
3. Confirm. The app appears in your launcher.

### Shortcut from inside the app

Open **Settings** (the dots icon in the top-right of the Library). Scroll to **Install** and tap **Install** — on supported browsers the OS prompt fires right there. On iOS the card shows the Share → Add to Home Screen reminder instead.

## Running it yourself

```bash
npm install
npm run dev        # http://localhost:5173
npm run build      # outputs dist/
npm run preview    # serves the production build
```

Push to `main` and GitHub Pages auto-deploys — no config needed.
